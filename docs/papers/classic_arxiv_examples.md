# Classic arXiv Examples

This list explains the 10 example-only paper method scaffolds bundled with ReproFlow.
They are included to help beginner users and AI agents understand how paper methods
can become model/config/trainer files. They are not completed reproductions.

| Example | arXiv paper | Main idea shown in ReproFlow |
| --- | --- | --- |
| `transformer_encoder_example` | [Attention Is All You Need](https://arxiv.org/abs/1706.03762) | self-attention over feature tokens |
| `bert_classifier_example` | [BERT](https://arxiv.org/abs/1810.04805) | CLS pooling plus auxiliary head |
| `resnet_mlp_example` | [Deep Residual Learning for Image Recognition](https://arxiv.org/abs/1512.03385) | residual blocks adapted to tabular features |
| `wide_deep_example` | [Wide & Deep Learning for Recommender Systems](https://arxiv.org/abs/1606.07792) | wide linear branch plus deep branch |
| `ncf_example` | [Neural Collaborative Filtering](https://arxiv.org/abs/1708.05031) | user/item interaction branches |
| `deepfm_example` | [DeepFM](https://arxiv.org/abs/1703.04247) | wide, FM, and deep branches |
| `dcn_example` | [Deep & Cross Network](https://arxiv.org/abs/1708.05123) | explicit cross layers plus deep branch |
| `autoint_example` | [AutoInt](https://arxiv.org/abs/1810.11921) | self-attentive feature interaction |
| `din_example` | [Deep Interest Network](https://arxiv.org/abs/1706.06978) | target-aware attention plus auxiliary interest head |
| `dlrm_example` | [DLRM](https://arxiv.org/abs/1906.00091) | bottom MLP, feature interactions, top MLP |
